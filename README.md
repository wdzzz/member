# member 会员周期管理系统

## 📌 项目简介
轻量级会员周期管理工具，基于 PHP + SQLite3 开发，专为群晖 Web Station 环境优化，实现会员信息、有效期、状态的快速管理。

## 🖥️ 运行环境
- **部署载体**：群晖 NAS（需安装 Web Station）
- **PHP 版本**：推荐 7.4+，需启用 SQLite3 扩展
- **数据库**：SQLite3（本地文件存储，无需独立部署）

## 🚀 群晖部署步骤
1. **安装 Web Station**（套件中心搜索安装）
2. **上传项目文件**至群晖 Web 目录（如 `/volume1/web/member`）
3. **启用 SQLite3 扩展**  
   - Web Station → 脚本语言设置 → 编辑使用的 PHP 版本  
   - 在「扩展」中勾选 `pdo_sqlite` 和 `sqlite3` → 保存并重启 Web Station
4. **访问并初始化**  
   - 浏览器访问 `http://192.168.*.***:端口/init.php`  
   - 完成数据库初始化，输入管理员账号和密码

> 系统仅支持单管理员账号，无普通用户权限划分，适合小型团队/个人使用。

## ✨ 核心功能
- **会员管理**：增删改会员信息（姓名、联系方式、类型、开通/到期时间）
- **周期自动计算**：根据开通时间+周期长度自动计算到期日
- **状态标识**：自动区分“正常”“即将到期（7天内）”“已过期”，并高亮显示
- **外部查询**：按会员名精准查询，供外部人员查询会员周期
- **数据本地化**：所有数据存储在 SQLite3 数据库文件 `member.db` 中

## ⚠️ 注意事项
- **备份**：定期备份 `member.db` 文件，避免数据丢失
- **权限**：建议仅内网访问，Web 目录权限设置为管理员可修改
- **PHP 兼容性**：推荐 PHP 7.4 ~ 8.0，避免使用过高版本

## 📝 总结
本系统是适配群晖 Web Station 的轻量会员周期管理工具，核心依赖 PHP + SQLite3，部署关键为启用 SQLite3 扩展；功能覆盖会员基础管理、周期自动计算与状态标识，满足小型场景的日常管理需求。
